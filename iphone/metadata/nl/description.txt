- Respecteert je privacy
- Spaart je batterij

Bij Organic Maps geloven we dat privacy een fundamenteel mensenrecht is:

- Organic Maps is een indie gemeenschap-gedreven project
- De code is open-sourced
- De kaartgegevens zijn afkomstig van OpenStreetMap, ook een community-gedreven project
- Wij beschermen uw privacy tegen de nieuwsgierige ogen van Big Tech.
